# 🎮 Simon Says Game

A simple **Simon Says Game** built with **HTML, Tailwind CSS, and JavaScript**. Test your memory by repeating the sequence of colors as it gets longer with each level.

## 🌐 Live Demo

🔗 **Play Here:** https://soumyodas001.github.io/Simon-Says-Game/

## ✨ Features

- Random color sequence generation
- Progressive difficulty
- Flash animations
- Score display on game over
- Easy restart

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS
- JavaScript (ES6)

## 🚀 How to Play

1. Press any key or click anywhere to start.
2. Watch the flashing color sequence.
3. Repeat the sequence by clicking the colored buttons.
4. Keep going until you make a mistake.
